ABOUT THE CODE (main.cpp)
=========================

Everything runs from a single file: main.cpp.


CODE SECTIONS
-------------


1. MAIN MENU
------------
Handles:
- Game selection
- Running selected game
- Returning to menu


2. SNAKE AND FRUIT
------------------
Contains:
- Snake movement logic
- Fruit generation
- Collision detection
- Score system


3. TIC TAC TOE
--------------
Contains:
- User input
- Bot logic
- Board drawing
- Win/draw detection


4. MINESWEEPER
--------------
Handles:
- Grid setup
- Mine placement
- Reveal system
- Flagging logic
- Win/Loss check


5. UTILITY FUNCTIONS
--------------------
Includes:
- Screen clearing
- Random number generator
- Simple delay
- ASCII banner display


WHY SINGLE FILE?
----------------
- Easy to compile anywhere
- Simple for beginners
- Good for college projects
- No header file management


CODING STYLE USED
-----------------
- Procedural programming
- Separate functions for each module
- GetAsyncKeyState for snake controls
- Logic-driven bot
- Reveal system similar to BFS
