ARCADE GAME PROJECT

Description:
A C console application with 3 classic games:
- Snake & Fruit
- Tic Tac Toe
- Minesweeper
Play directly in the console with interactive controls and real-time gameplay.

Files in the Project:
1. main.cpp - Main source code containing all 3 games
2. main.exe - Compiled executable (optional)

Features:
1. Snake & Fruit: Grow the snake, eat fruits, and track your score
2. Tic Tac Toe: Play against a bot with winning and defense logic
3. Minesweeper: 10x10 grid, place flags, avoid mines
4. Menu-driven interface for easy game selection
5. Real-time keyboard input for Snake
6. Interactive console output for all games

How to Run:
1. Compile main.cpp using any C/C++ compiler
2. Run the executable
3. Follow on-screen instructions to play

Notes:
1. Snake uses W, A, S, D keys for movement
2. Tic Tac Toe & Minesweeper require row and column input
3. Keep main.cpp and main.exe in the same folder
4. Enjoy classic games in console mode
