1. ARCADE GAME PROJECT
Description:
   A C console application with 3 classic games: Snake & Fruit, Tic Tac Toe, and Minesweeper.
   Play directly in the console with interactive controls and real-time gameplay.

2. Files in the Project:
   2.1 main.cpp    - Main source code containing all 3 games
   2.2 main.exe    - Compiled executable (optional)

3. Features:
   3.1 Snake & Fruit: Grow the snake, eat fruits, and track your score
   3.2 Tic Tac Toe: Play against a bot with winning and defense logic
   3.3 Minesweeper: 10x10 grid, place flags, avoid mines
   3.4 Menu-driven interface for easy game selection
   3.5 Real-time keyboard input for Snake
   3.6 Interactive console output for all games

4. How to Run:
   4.1 Compile main.cpp using any C/C++ compiler
   4.2 Run the executable
   4.3 Follow on-screen instructions to play

5. Notes:
   5.1 Snake uses W, A, S, D keys for movement
   5.2 Tic Tac Toe & Minesweeper require row and column input
   5.3 Keep main.cpp and main.exe in the same folder
   5.4 Enjoy classic games in console mode
