GAMEPLAY FLOW
=============

1. Main menu shows 3 games.
2. User selects a game.
3. Game runs inside console.
4. After finishing, user returns to menu.
5. User can play again or exit.


SNAKE GAMEPLAY
--------------
- Snake moves automatically.
- Fruit appears randomly.
- Score increases when fruit is eaten.
- Collision with wall or tail ends the game.


TIC TAC TOE GAMEPLAY
--------------------
- User enters row and column.
- Bot checks:
  1. Can it win?
  2. Can player win? -> block
  3. Otherwise random move
- Game ends with win, loss, or draw.


MINESWEEPER GAMEPLAY
--------------------
- User enters coordinates + action.
- Numbers show nearby mines.
- Flag bombs.
- Hitting a mine ends the game.
- Flag all mines to win.
